## Status

## Overview
This research proposes an explainable deep Graph Neural Network (GNN) framework for multi-stage Hepatitis C classification using patient similarity modeling.

The framework focuses on:
- Patient similarity graph construction
- Multi-stage disease classification
- Explainable AI for healthcare prediction
- Robust graph-based representation learning

## Research Contributions
- Developed an explainable GNN-based healthcare prediction framework
- Utilized patient similarity modeling for graph construction
- Addressed multi-stage Hepatitis C classification
- Improved interpretability in medical AI systems
